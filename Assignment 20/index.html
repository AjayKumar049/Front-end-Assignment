Closures Definition

A Closures is a function having acces to the parent scope. It preserve the data from outside.

For every closure we have three scopes.

  1. Local Scope (Own Scope)
  2. Outer Function Scope
  3. Global Scope

Examples On Closures in Javascript.

function show(){
var j = "J a local var of Outer Function";
document.write(j);
function innerFun () {
var k = "K a local var of Inner Function";
document.write(k);
document.write(j);
}
innerFun()
}
show();
OUTPUT: J a local var of Outer Function
K a local var of Inner Function
J a local var of Outer Function
Note : Outer Function Doesn't acces innerFun.
